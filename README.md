# Ecogarden
É um projeto acadêmico com o propósito de _irrigação inteligente_, para este desafio utilizamos IoT (***Internet of Things or Internet das Coisas***), esse repositório reune todos os repositórios do projeto, Front-End, Back-End e IoT.
